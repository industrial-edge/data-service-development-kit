ExampleClient - Data Service Development Kit

The ExampleClient of the Data Service Development Kit implements the following REST API. The different endpoints can directly be accessed by clicking at the route names. You immediately see the JSON response in the browser.

For routes which require an assetId or a variableId in the path initially you will see the error JSON response. Replace the :assetId or :variableId placeholder in the browsers address bar with the id of an existing object and execute the endpoint again.

Assets

Read the configuration of all existing assets.

Assets/:assetId

Read the configuration of a single asset selected via its id.

Variables

Read the configuration of all existing variables.

Variables/:variableId

Read the configuration of a single variable selected via its id.

Variables/:variableId/DataOfLastHour

Read the raw timeseries data of a single variable selected via its id. The data is read from the last hour. This example demonstrates the usage of the Data API of the Data Service.

Variables/:variableId/AverageOfLastHour

Calculate the average of a single variable selected via its id. The average is calculated over the last hour. The requested timerange is aligned to full minutes. This example demonstrates the usage of the Calculate API of the Data Service.

Variables/:variableId/AverageTrendOfLastHour

Calculate the average trend of a single variable selected via its id. The average trend is calculated over the last hour. The requested timerange is aligned to full minutes. The calculation period is 10 minutes. This example demonstrates the usage of the CalculateTrend API of the Data Service.

Variables/:variableId/LatestValue

Read the latest value of a single variable selected via its id. This example demonstrates the usage of the Data/Delta API of the Data Service.

Variables/:variableId/Monitor

Read cyclically each 5 seconds all new values of a single variable selected via its id. The execution is running in the background and the values are logged to the console. This example demonstrates how to run background tasks and how to call the API of the Data Service without the credentials of the interactive user.

Variables/StopMonitor

Stop the background task started with Monitor.
